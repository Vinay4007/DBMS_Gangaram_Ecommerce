## **How to Run?**
***To run the project follow the steps below:*** <br>
- Do git clone of this repository using the command `git clone `repolink``. <br>
- After doing git clone open two terminals in one terminal go to the frontend by using the command `cd frontend` and in the other terminal go to the backend by using the command `cd backend`. <br>
- Now in both terminals `install nodemodules` by using the command `npm install`. <br>
- After installing the nodemodules create a .env file and place it in the root folder. <br>
- provide portnumber , Mongodb Database Link , information in the .env file. <br>
- Now open new terminal and stay in the root directory and do data import by using the command `npm run data:import`. <br>
- Now after importing the data use the command `npm run dev` in the root directory to run the project. <br>
